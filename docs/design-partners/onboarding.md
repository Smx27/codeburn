# Design Partner Onboarding

Welcome to the AIInsight Design Partner Program.

## Overview

As a design partner, you'll help shape AIInsight by providing feedback on early features and helping us understand your AI coding workflow.

## What You Get

- **Early access** to new features
- **Direct line** to the development team
- **Priority support** for issues
- **Influence** on product roadmap
- **Free access** during the design partner period

## What We Ask

- **Regular feedback** on features you use
- **Bug reports** when you encounter issues
- **Feature requests** for improvements
- **Usage data** (anonymized) to help us understand patterns
- **Case study participation** (optional)

## Getting Started

### Step 1: Install AIInsight

```bash
# macOS
brew install aiinsight/tap/aiinsight

# Linux
curl -fsSL https://get.aiinsight.dev/install.sh | bash

# Windows
winget install AIInsight.CLI

# Or npm
npm install -g aiinsight
```

### Step 2: Create Account

1. Go to [app.aiinsight.dev/register](https://app.aiinsight.dev/register)
2. Enter your details
3. Verify your email

### Step 3: Generate API Key

1. Navigate to **Settings → API Keys**
2. Click **Generate API Key**
3. Copy the key

### Step 4: Connect Agent

```bash
aiinsight login
# Paste your API key when prompted
```

### Step 5: Start Syncing

```bash
aiinsight sync
```

### Step 6: View Dashboard

Open http://localhost:3000/dashboard to see your analytics.

## Providing Feedback

### Bug Reports

When you find a bug, please include:

1. **What you were doing**
2. **What you expected**
3. **What actually happened**
4. **Screenshots** (if applicable)
5. **Environment:**
   - OS and version
   - AIInsight version (`aiinsight --version`)
   - Browser (for dashboard issues)

### Feature Requests

When you have an idea, please include:

1. **The problem** you're trying to solve
2. **Your current workaround**
3. **Your proposed solution**
4. **How important** is this to you?

### General Feedback

For general feedback:

- What do you love about AIInsight?
- What's frustrating?
- What's missing?
- What would make you recommend it to others?
