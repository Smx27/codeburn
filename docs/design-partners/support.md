# Support

Getting help as a design partner.

## Overview

Design partners receive priority support. This document describes how to get help.

## Support Channels

### Slack (Fastest)

**Channel:** #aiinsight-design-partners

**Response time:** < 4 hours during business hours

**Best for:**
- Quick questions
- Urgent issues
- Real-time discussions
- Team collaboration

**How to use:**
1. Join the channel (invite in onboarding email)
2. Post your question or issue
3. Tag @team for urgent matters
4. Use threads for discussions

### Email

**Address:** design-partners@aiinsight.dev

**Response time:** < 24 hours

**Best for:**
- Detailed issues
- Sensitive matters
- Formal requests
- Off-topic discussions

**How to use:**
1. Send email with descriptive subject
2. Include relevant context
3. Attach screenshots if helpful
4. Allow 24-hour response

### GitHub Issues

**Repository:** https://github.com/getagentseal/codeburn/issues

**Response time:** < 48 hours

**Best for:**
- Bug reports
- Feature requests
- Documentation issues
- Technical discussions

**How to use:**
1. Create new issue
2. Use issue template
3. Include all required info
4. Label appropriately

### Weekly Check-ins

**When:** Every Thursday at 10 AM PT

**Best for:**
- Deep discussions
- Demo requests
- Roadmap questions
- Relationship building

**How to use:**
1. Add agenda to shared doc
2. Join Zoom call
3. Take notes
4. Follow up on action items

## Emergency Support

### Critical Issues

For system down or data loss:

1. **Slack:** Post in #aiinsight-design-partners with @channel
2. **Email:** design-partners@aiinsight.dev with "URGENT" in subject
3. **Phone:** Call support line (provided in onboarding email)

### Response Time

- **Critical:** 4 hours
- **High:** 24 hours
- **Medium:** 1 week
- **Low:** 2 weeks

## Common Issues

### Installation

| Issue | Solution |
|-------|----------|
| `aiinsight: command not found` | Add npm bin to PATH |
| `Permission denied` | Fix npm permissions |
| `better-sqlite3` build fails | Install build tools |

### Login

| Issue | Solution |
|-------|----------|
| "Invalid credentials" | Check email/password |
| "Account locked" | Wait 15 minutes |
| "Email not verified" | Check spam, resend |

### Sync

| Issue | Solution |
|-------|----------|
| "No sessions found" | Check provider detection |
| "Connection refused" | Check network |
| "Rate limit exceeded" | Wait and retry |

### Dashboard

| Issue | Solution |
|-------|----------|
| "No data available" | Run sync first |
| "Charts not loading" | Clear browser cache |
| "Page not found" | Check URL |

## Escalation

### When to Escalate

- Issue not resolved in expected time
- Critical impact on your work
- Multiple people affected
- Security concern

### How to Escalate

1. **Slack:** Tag @team-lead in #aiinsight-design-partners
2. **Email:** Send to design-partners@aiinsight.dev with "ESCALATION" in subject
3. **Phone:** Call support line (for critical issues only)

### Escalation Process

1. **Acknowledge:** We confirm receipt within 1 hour
2. **Investigate:** We investigate immediately
3. **Communicate:** We provide regular updates
4. **Resolve:** We fix the issue
5. **Follow-up:** We ensure you're satisfied

## Feedback

### About Support

After each support interaction, we'll ask:

1. Was your issue resolved?
2. Was the response time acceptable?
3. Was the support helpful?
4. Any suggestions for improvement?

### Improving Support

Your feedback helps us improve:

- We review support metrics weekly
- We update documentation based on common issues
- We train team on new issues
- We improve response times
