# AIInsight Product Documentation

## What is AIInsight?

AIInsight is an open-source AI usage analytics platform for engineering teams. It tracks token usage, costs, and performance across 30+ AI coding tools including Claude Code, Codex, Cursor, Gemini CLI, and more.
